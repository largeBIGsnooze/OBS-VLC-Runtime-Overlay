VLC Configuration

  1. Go to Preferences in VLC
  2. Check all on Show settings
  3. Go to Main interfaces and check Web
  4. Set your Host, Password and Port in the fields indicated below.
    5. Make sure the host, port and password coincides with the settings.json file.

OBS Configuration

  1. Add a Text Source and double click it
  2. Check the Read from file box and click Browse
  3. Select the movie.txt file in the logs/ directory.
